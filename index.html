Matthew's Simple Password Generator

Welcome to Matthew's simple password generator.
In order to obtain a secure password the generation algorithm must contain atleast 3 of the generation options listed below.
By default you are required to include the 26-character alphabet of lower-case letters and the 26-character alphabet of upper-case letters.
Next, you must choose up to 4 of the options for password generation by checking the boxes accordingly.
Lastly, you must choose the length of which the password will be, the minimum value for this is 10, the maximum value is 32.
Once done click the "Generate" button to populate the output.
